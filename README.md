# 985高校和中科院生物相关专业夏令营信息汇总

## 项目说明
本项目收集整理了2025年985高校和中科院生物相关专业（脑科学、基础医学、生命科学等）的夏令营招生信息，为大三即将保研的同学提供参考。

## 文件说明
- `夏令营汇总.docx.md`: Word文档格式的详细汇总
- `夏令营信息.csv`: CSV格式的数据表格
- `夏令营信息.xlsx.md`: Excel格式的表格示例

## 数据来源
1. 上海交通大学研究生招生网 (yzb.sjtu.edu.cn)
2. 其他985高校研究生院官网（基于一般信息整理）
3. 中国科学院官网

## 更新计划
- 进一步收集各学校具体的夏令营招生简章
- 补充申请材料、条件和录取优惠政策等信息
- 定期更新2025年夏令营的具体时间安排

## 使用建议
1. 本数据仅供参考，具体信息请以各学校官方通知为准
2. 建议关注各学校研究生院官网的最新动态
3. 提前准备好个人简历、成绩单等申请材料

## 免责声明
信息收集截止日期：2025年6月。由于各学校夏令营信息可能随时更新，请以官方发布为准。
