# 科研项目资料汇总

本项目包含以下两个主要方向的研究资料：

## 1. 夏令营信息收集

收集整理了2025年985高校和中科院生物相关专业（脑科学、基础医学、生命科学等）的夏令营招生信息，为大三即将保研的同学提供参考。

### 夏令营相关文件：
- `夏令营汇总.docx.md`: Word文档格式的详细汇总
- `夏令营信息.csv`: CSV格式的数据表格
- `夏令营信息.xlsx.md`: Excel格式的表格示例

## 2. GAD（谷氨酸脱羧酶）研究项目

谷氨酸脱羧酶（GAD）是一种重要的酶，能催化谷氨酸脱羧生成γ-氨基丁酸（GABA）。GABA是中枢神经系统重要的抑制性神经递质。

### GAD项目相关文件：
- `README.md`: 项目详细说明
- `gad_enzyme_miner/`: GAD酶挖掘工具包
- `real_gad_sequences/`: 真实的GAD序列数据库
- `gad_config.json`: 配置文件
- `requirements.txt`: Python依赖
- `setup.py`: 安装脚本

## 数据来源

### 夏令营信息：
1. 上海交通大学研究生招生网 (yzb.sjtu.edu.cn)
2. 其他985高校研究生院官网（基于一般信息整理）
3. 中国科学院官网

### GAD序列数据：
1. NCBI数据库
2. UniProt数据库
3. 专业文献中的序列信息

## 使用建议

### 夏令营信息：
1. 本数据仅供参考，具体信息请以各学校官方通知为准
2. 建议关注各学校研究生院官网的最新动态
3. 提前准备好个人简历、成绩单等申请材料

### GAD研究：
1. 适用于生物信息学和酶学研究
2. 可用于寻找具有特定功能的GAD酶变体
3. 可扩展用于其他酶家族的挖掘
