# Summer Camp Info & GAD Enzyme Research Project

## 项目概述

本项目包含两个主要方向的研究资料：

### 1. 夏令营信息收集
收集整理了2025年985高校和中科院生物相关专业（脑科学、基础医学、生命科学等）的夏令营招生信息，为大三即将保研的同学提供参考。

### 2. GAD（谷氨酸脱羧酶）研究项目
谷氨酸脱羧酶（GAD）是一种重要的酶，能催化谷氨酸脱羧生成γ-氨基丁酸（GABA）。GABA是中枢神经系统重要的抑制性神经递质。本项目包含GAD序列挖掘工具和相关数据库。

## 主要功能

### 夏令营信息
- 包含13个高校/研究所的夏令营信息
- 上海交通大学农业与生物学院、心理学专业、体育系夏令营详细信息
- 北京大学、清华大学、复旦大学、浙江大学等985高校夏令营
- 中国科学院脑科学与智能技术卓越创新中心、上海生命科学研究院夏令营

### GAD酶研究
- 谷氨酸脱羧酶（GAD）序列挖掘工具
- 真实GAD序列数据库（示例文件）
- 生物信息学分析方法
- 可用于寻找具有特定功能的GAD酶变体

## 使用说明

### 夏令营信息
1. 查看 `夏令营汇总.docx.md` 了解详细信息
2. 使用 `夏令营信息.csv` 进行数据分析
3. 关注各学校官网最新动态
4. 提前准备个人简历、成绩单等申请材料

### GAD酶研究
```bash
# 安装依赖
pip install -r requirements.txt

# 运行GAD挖掘工具
cd gad_enzyme_miner
python gad_miner.py --input real_gad_sequences/example.fasta --output results.json
```
