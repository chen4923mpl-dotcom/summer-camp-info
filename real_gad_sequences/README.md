# real_gad_sequences 目录

真实谷氨酸脱羧酶(GAD)序列数据库

## 文件说明
- `real_gad_sequences_example.fasta`: 真实的GAD序列示例文件
- `README.md`: 目录说明文档

## 数据来源
1. NCBI数据库
2. UniProt数据库
3. 专业文献中的序列信息

## 使用示例
```bash
python gad_miner.py --input real_gad_sequences_example.fasta --output output/results.json
```

## 注意事项
1. 所有序列均来自公开数据库
2. 序列信息仅供参考，需验证准确性
3. 使用时需遵守相关数据库的使用规定
