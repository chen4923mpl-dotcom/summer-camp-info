# 985高校和中科院生物相关专业夏令营招生信息收集计划

## 目标
收集2025年985高校和中科院生物相关专业的夏令营招生通知，包括：
1. 脑科学/神经科学
2. 基础医学
3. 生命科学
4. 生物医学工程
5. 生物技术

## 搜索策略
1. 直接搜索大学官网
2. 搜索教育部研究生招生网站
3. 使用学术论坛和社区信息
4. 查找各学校研究生院官方网站

## 需要搜集的关键信息
1. 学校名称
2. 学院/研究所
3. 专业方向
4. 夏令营时间
5. 申请截止日期
6. 申请要求
7. 报名方式
8. 联系方式

## 输出格式
1. Word文档汇总
2. Excel表格整理
3. GitHub仓库上传
